# Muhang 本地写作器

针对 `content/moments/` 的离线实时预览 Markdown 编辑器。

## 启动

在仓库根运行：

    npm run write

会自动：安装依赖 → 构建前端 → 启动后端(4747) 与 `hugo server`(1313) → 打开浏览器。

## 前置

- Hugo 0.135.0 extended（`hugo` 在 PATH 中）
- Node 18+

## 功能

- 新建项目：自动建 `content/moments/<N_slug>/index.md`；无文章时显示新建引导
- 表单管理：标题 / 心情(summary) / 日期 / 标签 / 封面(featured.*)
- 封面管理：点击封面图标上传；已有封面时弹出缩略图菜单，可更换或移除
- 工具栏插入：图片网格、大图、引用、阅读清单；插入后光标落在第一个待填位置，`Tab` 依次跳转，`Esc` 退出
- 图片面板：底部工具条切换，横向预览文章文件夹内的所有图片（新→旧）；在资源管理器里把图片放进文件夹、切回窗口即自动刷新；点击缩略图在光标处插入引用（网格内插 photo，否则插大图），已引用的图片文件名前打勾
- 拖图/粘贴入库：拖入或 Ctrl+V 粘贴图片（多图并行上传），自动复制进 bundle 并插入引用（截图自动命名为 `pasted-日期-时间`）；光标在 `{{< photos >}}` 网格内时插入网格图，否则插入大图
- 半 WYSIWYG 排版：标题分级放大、粗体/斜体/链接实时呈现，Hugo shortcode 淡化为等宽小字
- 自动保存（失败自动重试；关页/切后台时兜底保存）+ 右侧真实 Hugo 预览热重载（hugo server 掉线时显示占位并自动重连）
- 发布确认：弹层列出待提交文件、可编辑提交说明；检测并可一键清理未被正文引用的孤儿图片
- 标签自动补全：基于既有文章的标签
- 字数统计（底部工具条右侧，中文按字、英文按词）
- 专注模式：底部按钮或双击分隔条收起预览，编辑区限宽居中
- 快捷键：`Ctrl+S` 立即保存，`Ctrl+B` 加粗，`Ctrl+I` 斜体，`Ctrl+K` 插入链接
- 深/浅主题切换（顶栏按钮，或 `?theme=light|dark`；编辑器配色随主题联动），左右分栏可拖动，均记忆在 localStorage

## 说明

- 源 `index.md` 是唯一真相；frontmatter 的 YAML 注释在保存后不保留（字段值无损）。
- date 只在表单里改动了"日"时才被覆盖；手写的带时间/时区 date 原样保留。
